Product Search #

Full text search of Products and Keywords autocomplete. Those apis requires to have a search starter such as Open Search starter configured.

Search products #

Description:

Search products for a given Merchant Store and language.

Method:

POST

Url:

/api/v1/search

Body:

js
{
  "count": 20,
  "query": "shirt",
  "start": 0
}

Response:

status: OK

code: 200

Search keywords #

Description:

Autocomplete search.

Method:

POST

Url:

/api/v1/search/autocomplete?store=DEFAULT&language=en

Body:

js
{
  "count": 20,
  "query": "toto",
  "start": 0
}

Response:

status: OK

code: 200

js
{
    "values": [
        "Nike Sport Shirt",
        "T Shirt"
    ]
}
